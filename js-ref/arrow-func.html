# 箭头函数

# 1.没有this

  • 箭头函数没有this，需要通过查找作用域链来确定this的值
  • 不能使用call、apply、bind这些方法来改变this的指向

# 2.没有arguments

  • 箭头函数没有arguments参数
  • 想要获取箭头函数的参数，可以通过展开运算符rest参数的形式获取
let num = (...nums) => nums

# 3.不能作为构造函数

不能对其使用new命令

  • 没有construct属性（也没有call属性）
  • 没有原型（prototype）
let Foo = () => {}
new Foo() // TypeError: Foo is not a constructor
Foo.prototype // undefined

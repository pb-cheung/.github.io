# 遍历

# for...in

  • 遍历对象的可枚举属性（key-value中的key），适用于对象
  • 会将祖先中（Object.prototype）的可枚举属性也遍历了（可通过hasOwnProperty判断过滤解决）
  • 无法遍历Symbol类型（不可枚举）

# 获取Symbol的key值

Object.keys(obj).concat(Object.getOwnPropertySymbols(obj))

# for...of

ES6引入新特性

  • 适用于数组等可迭代对象（包括 Array，Map，Set，String，TypedArray，arguments 对象等等）
  • 迭代每次取值为元素值

举例：

var arr = [3, 5, 7];
arr.foo = "hello";
    
for (var i in arr) {
  console.log(i); // logs "0", "1", "2", "foo"
}
    
for (var i of arr) {
  console.log(i); // logs "3", "5", "7"
  // it doesn't log "3", "5", "7", "hello"
}

# 常用数组方法

# 增删

# 数组开头

  • 删除，shift()
  • 添加，unshift()

# 数组末尾

  • 添加，push()
  • 删除，pop()

这4个方法均会改变原数组

# 指定位置增删

  • splice(start, deleteCount, item0, item1, /* …, */ itemN)
    • 删除：下标为start的元素开始，删除deleteCount个（包含arr[start]）
    • 插入：插入从第三个参数开始的后续参数，把arr[start]挤到插入元素之后
    • 改变原数组

# 查找

# 返回元素

  • find(callbackFn(ele, index, array){})
    • 返回数组中满足测试函数的第一个值，无匹配的返回undefined

# 返回索引

  • findIndex(callbackFn(ele, index, array){})
  • indexOf(searchElement, fromIndex)
    • 相同点，返回匹配的第一个元素的索引，无匹配的返回-1
    • 不同点，参数不同

# 返回true/false

  • some(testFn(ele, index, array){})
    • 有匹配测试函数的元素则返回true
  • every(testFn(ele, index, array){})
    • 是否所有的数组元素都能通过测试函数，是返回true
  • includes(searchElement, fromIndex)
    • 包含返回true，不包含返回false

# 过滤

  • filter(testFn(ele, index, array){})
    • 返回新数组数，数组元素为满足测试函数的数组元素浅拷贝，不改变原数组
  • slice(start, end)
    • 下标从start开始，到end结束，不包含end，如果没有end则到数组末尾。
    • 返回的是浅拷贝，不改变原数组

# 迭代

  • forEach(callbackFn(ele, index, array){})
    • 返回undefined，不可链式调用，修改原数组
  • map(callbackFn(ele, index, array){})
    • 返回新数组，数组元素由回调函数返回值构成
  • reduce(callbackFn(accumulator, currentValue, currentIndex, array){}, initialValue)
    • 每次调用时，callbackFn的返回值都作为accumulator参数传递到下一次调用中，accumulator的最终值reduce()的返回值

# 排序

  • sort(compareFn(a, b){})
    • 不传参数，进行升序排序
    • 比较函数compareFn的返回值是数字（正数为升序，负数为降序），不是布尔值
    • 会改变原数组
  • reverse()
    • 反转数组，改变原数组

# 构造与转换

# 数组构造

  • Array(arrayLength)
    • 构造函数，用于创建数组Array('apple', 'banana')
    • 初始化一个两行三列的二维数组：new Array(2).fill(0).map(() => new Array(3).fill(0))
  • Array.from()
    • 是个静态方法（其他的方法一般是prototype上的）。创建一个从数字N开始，长度为L的连续递增数组Array.from({length: L}, (value, index) => index + N)
    • 从Map创建数组
    let map = new Map([
  ["1", "a"],
  ["2", "b"],
]);
let arr1 = Array.from(map.keys()); // ['1', '2']
let arr2 = Array.from(map.values()); // ['a', 'b']
  • fill(value, start, end)
    • 使用value填入索引从start到end的元素，如果end>length - 1，则到数组末尾截止不会继续添加（不会改变数组长度）
    • 会修改原数组
  • concat(value0, value1, ..., valueN)
    • 不改变原数组，返回新数组，元素是拷贝
    • 参数可以是数组也可以是普通字面量

# 字符串与数组转换

  • join(separator)
    • separator默认值为逗号“,”，不传此参数取值
    • 想要将数组转换为所有元素拼接arr.join('')
  • String.prototype.split(separator, limit)
    • 在字符串中每个separator处进行分隔，分隔得到的每个字符串作为数组元素，全部放入数组，并返回数组

# 作用域+this

# this

当前执行上下文（global、function 或 eval）的一个属性，在非严格模式下，总是指向一个对象，在严格模式下可以是任意值。

  • 全局上下文。即任何函数的外部，this指向全局对象，浏览器window、Node中globalThis；
  • 函数上下文。非方法调用，没有明确的调用主体的时候，this指向全局对象window，例如setTimeout、匿名函数等；
  • 方法调用。指向调用它的方法对象，与调用位置无关；
  • 构造函数。指向被构造的对象，即实例；
  • 箭头函数。指向箭头函数声明的时候，词法环境的this，跟调用的位置无关。
  • 修改指向。call/apply/bind，指向第一个参数；
  • 严格模式下。如果this没有被执行环境（execution context）定义，那 this是 为undefined；

# 函数上下文示例

(function(){
  console.log(this === window); // true
})()

setTimeout(function(){
  console.log(this === window); // true
}, 100)

# 箭头函数中this示例

// 创建一个含有 bar 方法的 obj 对象，
// bar 返回一个函数，
// 这个函数返回 this，
// 这个返回的函数是以箭头函数创建的，
// 所以它的 this 被永久绑定到了它外层函数的 this。
// bar 的值可以在调用中设置，这反过来又设置了返回函数的值。
var obj = {
  bar: function() {
    var x = (() => this);
    return x;
  }
};

// 作为 obj 对象的一个方法来调用 bar，把它的 this 绑定到 obj。
// 将返回的函数的引用赋值给 fn。
var fn = obj.bar();

// 直接调用 fn 而不设置 this，
// 通常 (即不使用箭头函数的情况) 默认为全局对象
// 若在严格模式则为 undefined
console.log(fn() === obj); // true

// 但是注意，如果你只是引用 obj 的方法，
// 而没有调用它
var fn2 = obj.bar;
// 那么调用箭头函数后，this 指向 window，因为它从 bar 继承了 this。
console.log(fn2()() == window); // true

# 题目一

以下代码给出打印结果：

var a = 10;

var obj = {
    a: 20,
    log0: function () {
        if (a) {
            a = 100;
        } else {
            a = 200;
        }
        console.log(a);
        console.log(this.a);
        var a = 1;
        console.log(a);
    },
    log1: function () {
        console.log(a);
    }
};

obj.log0();
obj.log1();

# 打印输出：

200
20
1
10

# 解释

  • 变量提升。第一个console之前，有逻辑修改打印值a。条件判断这里的a=undefined，因为log0函数内的var a = 1;是一个声明+赋值的语句，而var声明会被提前到函数内部最开始的地方而赋值不会，因此当前a已声明未赋值为undefined；
  • this指向。this.a中this指向obj，因此为obj.a为20；
  • 作用域。log0函数内已被声明的a，被赋值为1，打印1；
  • 作用域链。log1函数内没有定义变量a，沿着作用域链查找，找到了全局作用域的a变量，为10。

# 题目二

给出代码打印内容：

var a = 1;

var foo = { a: 2, show() { console.log(this.a); }}

var b = { a: 4 }

var show = foo.show;

show();
foo.show();
(b.show = foo.show)();

输出内容：

1
2
1

# 解释

  • this指向全局对象。全局变量show指向foo对象show函数，show()运行在全局作用域中，this指向window.a为1；
  • this指向调用对象。this指向foo对象。
  • this指向全局对象。b.show = foo.show这是一个赋值表达式，赋值表达式的结果为被赋值的函数本身，然后执行这个函数，因此现在我们执行的是foo对象的show方法，并且该方法在执行时内部的this将指向全局对象。

# 初始化涉及到的函数及其作用：

  • initComputed： 为每个computed属性创建一个对应的watcher——计算属性watcher
  • defineComputed：将computed属性挂载到Vue实例上（vm），并实现响应式，属性getter由createComputedGetter创建
  • createComputedGetter：创建并返回computed属性的匿名getter函数
  • 匿名getter函数：页面渲染过程，render函数被调用，_s(computed属性)调用到此函数。
    • 可以看到为了实现计算属性，Watcher新增了dirty、evaluate、depend等成员属性。

# watcher部分

  • dirty：
    • Observer构造函数中初始化为true——需要重新计算computed属性；
    • false——不需要重新计算，即"缓存"。（场景：一个组件内的模板中，多处插值使用该计算属性，gettr中不再重复计算）
  • evaluate：
    • 顾名思义——表达式求值，和dirty搭配使用，需要重新计算计算属性的时候调用此方法；
    • 调用observer.get，重置dirty（下次getter被调用就无需计算）
  • depend
    • 计算属性watcher依赖的属性的dep（本例中的：firstName的dep、lastName的dep），收集依赖

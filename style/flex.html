# flex布局

# 容器属性

  • flex-direction
    • row 从左到右
    • row-reverse 从右向左
    • column
    • column-reverse
  • flex-wrap
    • wrap
    • no-wrap
    • wrap-reverse
  • justify-content
    • flex-start
    • flex-end
    • space-between
    • space-around 每个子元素都有相同的距离（子元素间的距离是子元素与边框距离的两倍）
    • space-evenly 子元素间和子元素与边框的距离相同
  • align-items 区别于justify-content，它是垂直于子元素（项）的排列方向
    • flex-start
    • flex-end
    • center
    • stretch
    • base-line

# 项属性

  • order
    • 0 默认值，按照源码结构排列
    • number 数字小的在排列方向靠起始的一端
  • flex-grow 放大比例
    • number 按照数值比例，根据剩余空间，分配空间。默认值0
  • flex-shrink 缩小比例
    • number 按照数值比例，根据剩余空间，分配空间。默认值0
  • flex-basis 分配剩余空间之前，项目占据的空间
    • 百分比（基于项）
    • 5rem
    • auto
  • flex 以上三个属性的简写
  • align-self 项在垂直于排列方向的位置调节，可以覆盖容器align-items的设置
    • flex-start
    • flex-end
    • center
    • stretch
    • base-line

# order属性

.item {
    order: 1;
}

取值为整数（正负零都可以），默认为0，值小的排在上游，值相同的根据他们在源代码中的顺序排列。

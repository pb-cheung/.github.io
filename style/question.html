# 样式布局相关面试题

# 1. 如何实现一个自适应的正方形（正方形边长是屏幕宽度的10%）

# 方法1

.square {
  width: 10vw;
  height: 10vw;
}

缺点：vw存在兼容性问题

# 方法2

使用百分比内边距。

.square {
  width: 10%;
  margin-bottom: 10%;
  height: 0;
}

# 2.flex: 0 1 auto; 的含义

作用于flex元素的 Flex属性的简写形式。 将三个属性的数值依次排列。

flex: <flex-grow> [flex-shrink] [flex-basis]

使用了flexbox布局的区域叫flex容器，flex容器内的直系元素是flex元素。

  • flex-grow，取值：[0, ∞]，默认为0，主轴方向剩余空间分配规则，和兄弟节点按此值的比例分配；
  • flex-shrink，取值：[0, ∞]，收缩比例，当同容器内的其他flex元素尺寸超过容器时生效，大的数值可以比小数值的同级元素收缩程度更大；
  • flex-basis，flex元素的宽度，如果容器内给定了宽度就是指定值，如果没有给定就是元素内容宽度。

# 答案

flex: 0 1 auto;

对应的含义就是：

在这里 flex-grow 的值为 0，所以 flex 元素不会超过它们 flex-basis 的尺寸。 flex-shrink 的值为 1, 所以可以缩小 flex 元素来防止它们溢出。 flex-basis 的值为 auto. Flex 元素尺寸可以是在主维度上设置的，也可以是根据内容自动得到的。

# 4.CSS呼吸效果的动画

  • animation-anme: OpacityBreath; // 关键帧设置的名称
  • animation-duration: 2s; // 持续时间
  • animation-timing-function: ease-in-out; // 设置动画速度，即通过建立加速度曲线，设置动画在关键帧之间是如何变化。
  • animation-iteration-count: infinite; // 动画的重复次数。无限循环

    animation: OpacityBreath 2s ease-in-out infinite;

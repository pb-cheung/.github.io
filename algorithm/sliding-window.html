# 滑动窗口

sliding window

# 一、题目

力扣 209 长度最小的子数组 (opens new window)

给定一个含有 n 个正整数的数组和一个正整数 target 。
找出该数组中满足其总和大于等于 target 的长度最小的 连续子数组 [numsl, numsl+1, ..., numsr-1, numsr] ，并返回其长度。如果不存在符合条件的子数组，返回 0 。
示例 1：
输入：target = 7, nums = [2,3,1,2,4,3]
输出：2
解释：子数组 [4,3] 是该条件下的长度最小的子数组。
示例 2：
输入：target = 4, nums = [1,4,4]
输出：1
示例 3：
输入：target = 11, nums = [1,1,1,1,1,1,1,1]
输出：0

# 二、暴力解法

  • 双重循环，外层循环控制开始元素，内层为结束元素，遍历获得满足条件的所有子数组，取其中值最小的
  • 时间复杂度O(n²)

# 三、滑动窗口解法

  • 滑动窗口，一层循环，让窗口能“滑动”起来。时间复杂度O(n)
  • 关键点：循环索引值是窗口起始还是结束位置，应该是结束位置
  • 循环开始，窗口扩大新增一个元素，就累加获取和
    • 和大于等于目标值时，就得到了一个可能满足条件的子数组，将其长度与现有长度作对比，取小的值
    • 我们的目的是找到符合条件且最短的子数组，此时需要移动窗口左边界，缩小窗口。又一个关键点，左边界移动一次后的窗口内的所有元素还是符合条件的，此时需要继续移动左边界，因此，上一个小点的条件判断应该用while而不是if
/**
 * @param {number} target
 * @param {number[]} nums
 * @return {number}
 */
var minSubArrayLen = function(target, nums) {
    const len = nums.length;
    let pLeft = 0;
    let sum = 0;
    let result = len + 1;
    for (let pRight = 0; pRight < len; pRight ++) {
        sum += nums[pRight];
        while (sum >= target) {
            const subLen = pRight - pLeft + 1;
            result = Math.min(result, subLen);
            // 移动左边界
            sum -= nums[pLeft];
            pLeft ++;
        }
    }
    if (result === len + 1) {
        return 0;
    }
    return result;
};

# 排序算法

# 时间复杂度

算法的Time complexity是一个函数，它定性描述该算法的运行时间。时间复杂度用大O符号表述，它表述的是渐近的，即输入值大小趋近无穷时的情况。

假设计算机执行每个基本操作（语句、指令）的耗时是一致的，那么时间复杂度可以通过计算算法执行的基本操作的数量来估计，算法所花费的时间和基本操作的数量最多差一个常数因子。

# 大O符号

大O符号（Big O notation），又称为渐进符号，是用于描述渐进行为的数学符号。它是用另一个（通常是更简单的，例如：n, n * logn ）函数来描述函数数量级的渐进上界。

# 常见的时间复杂度

# 1.常数时间

O(1)，没有循环等特殊结构的一些语句，无论多少行

# 2.线性时间

O(n)，单层循环结构

# 3.二次时间

O(n²)，双层循环结构

# 4.对数时间

O(logn)， 举例：

int i = 1;
while(i<n)
{
    i = i * 2;
}

在while循环中，每次都将i乘以2，乘完之后，i距离n就原来越近了。我们试着求解一下，假设循环x次之后，i就大于n了，这个循环就退出了，也就是说2的x次方等于n（i = i0 * 2 * 2 * 2 ...，共x次），那么x = ㏒ 2 n

# 5.线性对数时间

O(nlogn)，将时间复杂度为O(logn)的代码循环N遍的话，那么它的时间复杂度就是 n * O(logN)，也就是了O(nlogN)。 例如：

for(m=1; m<n; m++)
{
    i = 1;
    while(i<n)
    {
        i = i * 2;
    }
}

# 冒泡排序（Bubble Sort）

步骤：

  1. 比较相邻的两个元素，如果是逆序就调整它们的位置；
  2. 从序列开头的两个开始，重复进行1.的操作，这样一轮循环过后，最大的元素就被“冒泡”到序列末尾；
  3. 针对剩下的元素进行相同的操作，依次找到次大的，第三大的...，直到没有一对元素可以比较。
function bubbleSort( arr, n)
{
var i, j;
for (i = 0; i < n-1; i++)
{
    for (j = 0; j < n-i-1; j++) // 注意这里边界的判定条件n-i-1
    {
        if (arr[j] > arr[j+1])
        {
        swap(arr,j,j+1);
         
        }
    }
}
}

时间复杂度：O(n²)

# 优化

问题点：即使是一个已经排好顺序的一个序列，它仍然要小号O(n²)的时间。

优化：某次内层循环未发生元素交换的情况，可以结束外层循环（排序已完成）。

# 归并排序（Merge Sort）

该算法是分治法（Devide and Conquer）的一个应用，且各层分支递归可以同时进行。 分为两个部分： 分割：递归地把当前序列平均分割成两半。 集成：在保持元素顺序的同时，将上一步得到的子序列集成到一起（归并）。

步骤：

  1. 将待排序的序列不断拆分成两个子序列，递归调用，直到每个子序列只有一个元素为止；
  2. 从最小的相邻单个子序列开始，对子序列进行合并，合并过程中排序；
  3. 返回合并后的子序列，直到排序完成。
function merge(left, right){
  var result = [];
  while(left.length > 0 && right.length > 0){
    if(left[0] < right[0]){
      result.push(left.shift());
    }else{
      result.push(right.shift());
    }
  }
  return result.concat(left, right);
}

function mergeSort(arr){
  if(arr.length <=1) return arr;
  var middle = Math.floor(arr.length / 2);
  var left = arr.slice(0, middle);
  var right = arr.slice(middle);
  return merge(mergeSort(left), mergeSort(right));
}

时间复杂度：O(nlogn)

# 快速排序（Quick Sort）

步骤：

  1. 选取一个基准元素（pivot）
  2. 将序列中所有小于基准元素的元素放入左数组，大于等于基准元素的放入右数组；
  3. 对基准元素左右两个子序列递归地快速排序，并对结果进行拼接；
  4. 重复上述过程，知道子序列长度小于等于一，返回子序列
var quickSort = function(arr) {
  if (arr.length <= 1) {
    return arr;
  }
  var pivotIndex = Math.floor(arr.length / 2);
  var pivot = arr.splice(pivotIndex, 1)[0];
  var left = [];
  var right = [];

  for (var i = 0; i < arr.length; i++) {
    if (arr[i] < pivot) {
      left.push(arr[i]);
    } else {
      right.push(arr[i]);
    }
  }
  return quickSort(left).concat([pivot], quickSort(right));
};

时间复杂度：O(nlogn)

function bucketSort (input) {
    // 桶排序
    let arr = input.split("");
    let len = arr.length;

    const sorted = new Array(26).fill(""); // 创建桶

    for (let s of arr) {
        // 放入桶内
        if (/[a-z]/.test(s)) {
            sorted[s.charCodeAt() - "a".charCodeAt()] += s;
        } else if (/[A-Z]/.test(s)) {
            sorted[s.charCodeAt() - "A".charCodeAt()] += s;
        }
    }
    // 桶内无需再排
    let result = sorted.join("").split("");

    // 非字母元素
    for (let i = 0; i < len; i++) {
        if (/[^A-Za-z]/.test(arr[i])) {
            result.splice(i, 0, arr[i]);
        }
    }

    console.log(result.join(""));
}

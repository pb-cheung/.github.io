# 移除元素

# 一、题目

力扣27 移除元素 (opens new window)

给你一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素，并返回移除后数组的新长度。
不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并 原地 修改输入数组。
元素的顺序可以改变。你不需要考虑数组中超出新长度后面的元素。
示例 1：
输入：nums = [3,2,2,3], val = 3
输出：2, nums = [2,2]
解释：函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2。你不需要考虑数组中超出新长度后面的元素。例如，函数返回的新长度为 2 ，而 nums = [2,2,3,3] 或 nums = [2,2,0,0]，也会被视作正确答案。
示例 2：
输入：nums = [0,1,2,2,3,0,4,2], val = 2
输出：5, nums = [0,1,4,0,3]
解释：函数应该返回新的长度 5, 并且 nums 中的前五个元素为 0, 1, 3, 0, 4。注意这五个元素可为任意顺序。你不需要考虑数组中超出新长度后面的元素。

很明显不能使用splice、slice之类的库函数，对空间有要求，就是给了一个元素的空间

# 二、暴力解法

暴力解法，笨方法，双重循环。

  • 第一层循环遍历所有元素，跟目标元素比较
    • 如果相等，就从下一个元素开始依次向前移动一个位置（以此来实现删除）
    • 如果不相等，就继续遍历
  • 第二层循环就是做这个移动操作的，要注意移动完成后，外层循环不能直接继续遍历，要判断下移动过来的元素是不是也是需要删除的
/**
 * @param {number[]} nums
 * @param {number} val
 * @return {number}
 */
var removeElement = function(nums, val) {
    let temp;
    let count = 0;
    let len = nums.length;
    for (let i = 0; i < nums.length - count;) {
        if (nums[i] === val) {
            count ++;
            for (let j = i; j < nums.length - 1; j++) {
                nums[j] = nums[j + 1];
            }
        } else {
            i ++;
        }
    }
    return len - count;
};

# 三、双指针法

明确两个指针的含义，指向

  • 快指针，用来遍历数组，指向当前检查的元素，从0开始
  • 慢指针，用来指向目标数组（删除目标元素后的数组）的最后一个元素的下标，从0开始，
    • 遇到非要删除元素，将元素赋给当前慢指针对应的下标，慢指针自增一次
    • 遇到要删除的元素，它不是我们新数组需要的，慢指针不做操作
/**
 * @param {number[]} nums
 * @param {number} val
 * @return {number}
 */
var removeElement = function(nums, val) {
    let pFast = 0;
    let pSlow = 0;
    for (; pFast < nums.length; pFast++) {
        if (nums[pFast] !== val) {
            nums[pSlow] = nums[pFast];
            pSlow++;
        }
    }
    return pSlow;
};
